# 🛒 Ecommerce Mock Store

Live Demo: https://ecommerce-mock-delta.vercel.app/
A modern, responsive e-commerce mock built to simulate a real online store experience.  
This project focuses on conversion-oriented UI, clean components, and scalable frontend architecture.
It was created as a hands-on practice project to strengthen my skills in e-commerce development and Shopify-like storefronts.

---

## 🚀 Features

- Product listing page
- Product detail view
- Shopping cart
- Add/remove items
- Coupon/discount input
- Responsive mobile-first design
- Clean UI with TailwindCSS
- Component-based architecture
- Ready for API or payment integrations

---

## 🧠 Tech Stack

- React
- JavaScript
- Tailwind CSS
- Vite
- Local state management
- Deployed with Vercel

---

## 🎯 Goals of this project

- Practice real-world e-commerce flows
- Build reusable UI components
- Improve conversion-focused layouts
- Simulate Shopify storefront behavior
